# Welcome To Hazard!

The famous hacker organization Cipher Society was compromised and their data was leaked.
Hazard allows you to experience their secrets through a social media user interface!



## Prerequisites

Before you get started, make sure you have the following requirements in place:

- [Node.js](https://nodejs.org/en) (v18.0.0 or v20.0.0+) - Verify by running `node --version`
- [npm](https://www.npmjs.com/) (v9.0.0+) - Verify by running `npm --version`

## Setup

1. Clone the repository
    ```bash
    git clone https://github.com/eskilschjong/hazard.git
    cd hazard
    ```
2. Install dependencies
    ```bash
    npm install
    ```
3. Start the dev server
    ```bash
    npm run dev
    ```
