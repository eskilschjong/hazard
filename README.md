# Welcome To Hazard!

The famous hacker organization Cipher Society was compromised and their data was leaked.
Hazard allows you to experience their secrets through a social media user interface!


## Tools & Dependencies

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
- React Router

## Data Sources

- Posts, users and comments - https://jsonplaceholder.typicode.com
- Images - https://placehold.co
